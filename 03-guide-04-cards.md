# 对话区卡片指引

> 这份文档说明 AI Demo Shell 左侧对话区当前稳定支持的卡片能力，以及未来交互组件的边界。

---

## 当前稳定卡片：生成物卡片

生成物卡片用于提示用户右侧工作台已有内容可查看。用户点击卡片后，右侧工作台打开对应步骤的生成内容。

当前生成物卡片只支持：

- 图标
- 标题
- 说明
- 点击打开右侧工作台

当前暂不支持：

- 状态文案
- 多个操作按钮
- 表单输入
- 展开折叠
- 列表明细
- 图片预览

---

## 生成物卡片如何出现

生成物卡片不需要单独配置。只要某个步骤的 `### 右侧面板` 中有实质内容，对话区就会自动出现一张生成物卡片。

如果右侧面板为空，或者写的是「暂无内容」这类占位文字，生成物卡片不会出现。

推荐在右侧面板开头写：

```markdown
### 右侧面板

**工作台卡片：智能审核报告**

查看合同、附件、金额、签章、账户一致性等预审结果。

**预审报告摘要**

| 指标 | 结果 |
|---|---|
| 预审结论 | 部分通过 |
| 主要风险 | 收款账户与合同备案账户不一致 |
```

其中：

- `智能审核报告` 会成为生成物卡片标题。
- 第二行说明会成为生成物卡片描述。
- 后续内容会在右侧工作台展示。

## 左侧未来交互组件白名单

后续如果 shell 已开发对应组件，左侧可以扩展：

| 组件 | 用途 | 典型场景 |
|---|---|---|
| 单选卡片组 | 从多个互斥选项中选择一个 | 选择金额口径、选择催办方案 |
| 多选卡片组 | 从多个选项中选择多个 | 勾选补充附件、选择审核维度 |
| 基础表单 | 输入少量结构化字段 | 修改收款账户、填写联系人 |
| 基础表单 - 输入 | 输入文本、金额、日期等 | 填写申请说明、修改账号 |
| 基础表单 - 单选 | 选择一个固定枚举值 | 款项类型、是否加急 |
| 基础表单 - 多选 | 选择多个固定枚举值 | 附件类型、通知对象 |

在组件未开发前，不要在 `business-design.md` 中假定这些组件可用。
