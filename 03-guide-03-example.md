# 场景说明

施工单位资料员在发起工程进度款付款申请前，需要核对合同、金额、附件、收款账户和审批路径。成本请款专员会识别付款资料，检查缺失附件，提示金额差异，生成预审报告，并协助提交审批和跟进进度。

# 常见追问

## 为什么推荐用明细表金额？

汇总表中的安装工程进度款 560,000 元没有在明细表中列示，缺少计量依据。若按汇总表提交，成本复核大概率会退回要求补充明细。按明细表 **2,000,000 元** 先提交，可以降低退回风险。

## 收款账户不一致还能提交吗？

可以提交，但需要附带账户差异说明。当前风险为提醒项，不是阻断项。更稳妥的做法是先补充账户变更证明，或把付款账户改回合同备案账户。

## 预计多久到账？

按当前审批链和历史平均耗时，预计 2026-05-20 到账。若项目成本复核或财务复核超期，到账时间会顺延。

## 与任务无关的问题

我专注于工程付款申请资料识别、智能预审和审批跟进。这个问题超出当前任务范围，如需继续处理付款申请，我可以帮你核对金额、附件或审批进度。
