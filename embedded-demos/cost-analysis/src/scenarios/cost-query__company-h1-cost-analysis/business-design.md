# 场景说明

成本管理人员需要快速了解公司层面的成本结构，判断建安、土建、安装、精装等科目的金额、建面单方和占比。成本问数会先让用户确认分析维度和核心指标，再生成一份可点击打开的成本构成分析报告。

# 常见追问

## 为什么“上半年”没有按时间筛选？

当前演示数据口径来自项目级成本指标汇总，未包含可直接筛选的清单价格时间区间。报告中已明确提示这一点，正式使用时需要补充具体时间范围。

## 为什么建筑安装工程费占比这么高？

当前数据范围主要是项目成本指标，建筑安装工程费覆盖土建、安装、公共部位精装、基础工程等主力科目，因此在总成本中占比最高。

## 土地成本为什么没有数据？

科目体系中存在土地成本，但当前演示数据未返回金额。常见原因是土地成本单独核算，或未纳入本次成本指标数据包。

## 这份报告能导出吗？

当前 Demo 以浏览器演示和右侧工作台查看为主，真实导出属于后续组件能力，不在本次原生演示范围内。

## 与任务无关的问题

我专注于公司、项目、业态、科目等成本数据分析，请继续围绕成本分析提问。
