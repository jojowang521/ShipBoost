# 组件缺口记录

## 本次 Demo 已支持

- 原生 / 独立模式打开演示页。
- 首页展示“成本问数”专员身份和“上半年成本分析”快捷入口。
- 左侧对话用单选表格展示分析维度和核心指标。
- 用户确认分析口径后继续生成报告。
- 生成物卡片展示“公司成本构成分析报告”。
- 点击生成物卡片后，右侧工作台打开完整成本分析报告。

## 未实现的真实业务能力

| 能力 | 当前处理 |
|---|---|
| 真实成本数据库查询 | 使用固定模拟数据演示分析链路 |
| 真实时间筛选 | 在报告中说明当前数据未按“上半年”筛选 |
| 报告导出 | 作为后续组件能力，不在本次演示范围内 |
| 复杂筛选表单 | 用对话区单选表格模拟关键分析口径选择 |
