# 配置侧边栏模式

侧边栏模式用于“产品页面 + 右下角 AI 入口 + 右侧 AI 侧边栏”的演示。

## 什么时候使用

用户明确要求：

- 嵌入产品页面；
- 点击 AI 入口打开；
- 侧边栏；
- 右侧 AI 助手；
- 在已有产品 demo 上接入 AI。

其他情况默认使用原生 / 独立模式。

## 预览方式

```text
?shell=sidebar
```

可选宿主页面：

```text
?shell=sidebar&host=list
?shell=sidebar&host=workbench
?shell=sidebar&host=detail
?shell=sidebar&host=split
```

## 行为规则

- AI 入口默认在浏览器右下角，支持上下拖动并记住位置。
- 点击入口打开右侧侧边栏。
- 关闭只隐藏侧边栏，不结束当前对话。
- 新建会话才重置当前对话。
- 点击生成物卡片后，工作台出现在聊天侧边栏左侧。
- 全屏时，如果没有打开工作台，聊天区居中并沿用原生模式最大宽度；如果已打开工作台，则工作台和聊天区整体最大化。

## 宿主页面预设

第一批支持：

- `list`：列表页，适合审核、查询、管理、清单、台账；
- `workbench`：工作台页，适合首页、待办、经营、看板、总览；
- `detail`：详情页，适合详情、单据、合同、项目、申请单；
- `split`：分栏页，适合项目树、分类、组织树、左右联动。

宿主页面数据必须贴合业务场景，不写“示例数据”“测试项目”“暂无内容”。
